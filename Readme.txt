Ability to recognize of 4 door signs( professor Echtler, Rodehorst, Hornecker, Wuthrich) is implemented to this software.
When application is launched, it will catch web pages of aforementioned professors from university website. After loading
those web pages by WebView, it will save those pages as a bitmap object. So, when software recognize a sign, it will show
related information to user.
